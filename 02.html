로컬에서 commit 하면 버전이 생성됨
    로컬의 파일을 원격저장소로 올리기
    1- 깃허브에 로그인
    2- 저장소 생성
    3- 로컬(master)과 원격(main) 연결
    git branch -M main
    git remote add origin https://github.com/parkseoan/git-test.git
    git push -u origin main
